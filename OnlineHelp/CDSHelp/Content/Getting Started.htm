Getting Started

Welcome to CDS. Once you have logged into CDS, you can see the functions relating to your user profile on your Home page. Functions are grouped into the following categories:

  • Operational (operational users only)
  • Application (administrative users only)
  • Management
  • Report

The functions available may vary, depending on your organization's configuration.

Getting started with CDS

  1. Open your browser and type the CDS URL provided to you by your administrator.
  2. Enter your user name and password, then click Log in. CDS displays your Home page with the functions available to your user type.
  3. Navigate the functions using the links on your Home page. Click the Home link at the top of the page at any time to return to the Home page.
  4. You can log out using the Log out link at the top right of your Home page.

Changing your password

You can change your CDS password at any time using the Change password link at the top right of your Home page.

CDS online help

To get help for a specific CDS function, click Help at the top right of the page. Clicking Help from the Home page takes you to the help for all functions.

You can navigate the help using the Quick links or the menu on the left. The menu on the left is grouped by user type, giving easy access only to the help for the functions of your own user type. To locate a help topic by keyword, use the search bar at the top.

There is no help available for specific report types. This is because reports are not standard in CDS and can be changed by the administrator.
