Manage rules for customs declarations

You use the Manage rules for customs declarations function to configure rules to automatically transfer declarations to either the local Customs authority or the designated postal operator, once the mail item has been stored in a final (not draft) state.

If a mail item complies with the automatic transfer rules defined, CDS does one or both of the following:

  • generates a send to customs mail item event, triggering the generation of EDI message CUSITM (provided a valid EDI exchange has been configured for local Customs)
  • generates a send to destination post mail item event, triggering the generation of EDI message ITMATT (provided a valid EDI exchange has been configured for the destination postal operator)

To automatically send a declaration to Customs for inbound or outbound mail

  1. Click the Customs transfer rules tab.
  2. Select the required criteria from A to R in the outbound mail or inbound mail section.
  3. Select the Logical expression combining these criteria.

Tip: The ? sign can be used as a wild card (i.e. to specify any value).
