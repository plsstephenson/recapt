Check outbound item and create response at customs

To view the declaration for an outbound mail item

  1. Search for the mail item by entering the mail item id and clicking OK, then proceed to step 2 or 3, depending on what you want to do next.

    2. Tip: If you do not know the mail item id, use the Search declarations function instead (see Search declarations). You can then perform the following actions from there.

  2. Take an action on one or more mail items, as follows:
    1. Select an option in New action. The Take action link next to the mail items becomes visible.
    2. Select a Decision code.
    3. Click the Take action link next to the required mail item(s).
  3. Click the Select link next to the required mail item(s). After you have selected a declaration, you can view it and enter a response. By default, the postal declaration and Customs response are on different pages. To view them on the same page, update the default setting in Manage my preferences (see Manage my preferences).

To create a Customs response

  1. Click the Customs response tab.
  2. Select your decision from the Decision drop-down menu.
  3. Enter a reason for the decision. If there is duty to be paid, check the Dutiable check box and enter a Clearance date (if the item is cleared).
  4. Optionally, you can enter additional information:
    • Tax/duty information in the Total duties/taxes table; click Edit to add information and Update to save your changes (Delete deletes an entry, and Cancel cancels the current entry).
    • Enter the amount and currency code for any other fees (e.g. handling fees) in Total fee.

    Tip: Administrator users can also automate duties and tax ratings assessments by:

    • Configuring tax rules at the national level (only Customs Administrators can configure these rules); see Manage rules for customs responses for more information
    • Using the tax scripting function for more complex schemes (only Master/System Administrators can use this function)

  5. Once your tax rules are configured, click Calculate taxes to perform the calculation according to the configured rules.
  6. Send the response to the Post by:
    1. Selecting action 52 (Store final (automatic transfer)).
    2. Clicking Store.

Tip: You can also store a draft of your decision if elements of your decision are still missing, by selecting action 51 (Create/update draft).

Updating a decision

Customs can update a decision if information about a postal item has changed. Examples of changes are:

  • the risk assessment changes (to more or no risk)
  • duties and taxes have been prepaid.

For example, you can clear an item after receiving confirmation of prepayment charges by following the steps above.
