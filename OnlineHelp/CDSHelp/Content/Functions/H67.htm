Upload EDI files

The Upload EDI files function is for uploading EDI files to CDS (for example, ITMATT, CUSRSP, or even CUSITM). This functionality is intended mainly for testing purposes on the CDS central pre-production server. In a production setting, the EDI message flow is handled either by the PNG (Postal Network Gateway application) in a standalone installation, or directly by Post*Net in the CDS-Central version of the application.

To upload a file

  1. Click Browse to select a zip file containing either CUSITM, CUSRSP or ITMATT files to upload to the CDS database.
  2. Click Upload.

If the file was uploaded successfully, the message 'Your file was uploaded successfully' displays on your screen.

The fact that your zip file was successfully uploaded does not guarantee that all messages were correctly imported into the database. The CDS administrator can check this by reviewing the logs.
