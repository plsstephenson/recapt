DIY for administrators

User administration

You can find all the user administration tasks in the Administration menu:

  • User account and user role management
  • Access control rules
  • Access token management

Create a new user role

You can create roles to assign when you create new users.

  1. Select Roles.

  1. Click .

  1. Enter a name and description for the role and click Save. A new user role entry appears in the list.

  1. Click to enable permissions for the role. Click Save when you have finished.

Create a new user

  1. Select User.

  1. Click .

  1. Enter the email address and click Save. An account verification email is sent to the user and a new user entry appears in the list with a Pending verification icon next to it.

  1. After the user has verified the account, the user entry changes to Verified

  1. A new user token is automatically assigned to the user, which you can view in Access tokens.

  1. You can now update the user's security settings:
  • click to view a list of permissions and enable/disable them individually
  • click to assign a user role with all the associated permissions to the user account

Create access control rules

Once you have created your API, you can grant users restricted access to it, by limiting users/user roles to a specific type of access to a given data model or endpoint.

  1. Select Access control rules.
  2. Click .
  3. Select the model from the drop-down menu.
  4. Specify a method, if required.
  5. In Principal type, specify if this rule applies to a User or Role.
  6. In Principal id, specify the role or user name.
  7. Toggle the Permission button to switch the permission on.

Manage client application access

You can specify details of client applications which can access new APIs created in DIY.

  1. Select Client applications.
  2. Click .
  3. Enter the details of the client, including its name, the type of access granted, whether the client application is confidential or public, a list of scopes and callback URLs.
  4. Click Save.

Manage access tokens

New access tokens are created automatically when create a new user account. Select Access tokens to view or delete access tokens.
