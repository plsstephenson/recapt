Overview

The Web Tracking API allows you to retrieve events for one or more mail items scanned in IPS.

GET endpoint(s)

/Tracking (single mail item)

/Track (multiple mail items)

Resource URL

http://localhost/API/Tracking/{mail-item-id}/{language:[A-Za-z]2}/{full-view:bool}

http://localhost/API/Track/{comma-separated-list-of-mail-item-ids}/{language:[A-Za-z]2}/{full-view:bool}

Resource information

Response formats JSON, XML
Requires authentication? No
Rate limited?  
Requests / 15-min window  

Parameters

Path parameters:

Parameter Description Format Default value Mandatory
language ISO language code; when set, any reference data included in the response is returned along with the translation in the specified language (if it exists in the CT tables) [A-Za-z]2 Null No
full-view Defines whether signature images (converted to Base64) are returned. Set to True only if there is enough bandwidth to transfer large data packages. Not advised for mobile applications. Boolean False No
