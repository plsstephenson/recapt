    Introduction

    Welcome to IDPlatform API! You can use our API to manage access tokens for user authentication, including functions such as:

    • creating new access tokens
    • retrieving information on tokens and users
    • refreshing tokens which have expired

    Note! This is real API documentation with details changed for anonymity. None of the links will work.

    Authentication

    IDPlatform API uses API keys to allow access to the API. You can request a new API key on our QuickStart Page.

    IDPlatform API expects the API key to be included in the header when requesting a new access token.

    HowTo

    For more information on how to accomplish different access token tasks with IDPlatform, watch this video.
