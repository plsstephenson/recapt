Getting started

Welcome to the QSF application, a tool for UPU staff and organizations to manage and monitor Quality of Service Fund contributions, including:

  • organization information and terminal dues group membership
  • bilateral agreements
  • CN61 and CN64 forms
  • QSF reference data
  • QSF financial documents

User types

This help is only for stakeholder users.

Home page

When you log in to QSF, the Home page is displayed.

  • In the center of the page, QSF displays a horizontal navigation bar. The bar is divided into the key functions that you can perform:
    • Stakeholder management (the Home page displays this function with the name of the stakeholder)
    • CN61 / CN64
    • QSF Reference Data
    • QSF Financial Documents
  • In the top right corner of the page, you have access to general user functions (Profile management, Logout and the language menu). You can also click the Home icon to return to the Home page at any time.

Logging out

To log out from QSF, click the user menu at the top right of the screen and select Logout.

Accessing the online help

To view the online help for QSF, click ?.

From an open online help window, you can access other online help topics using one of the following methods:

  • Using the Table of contents
  • Using the Search Bar: enter a keyword or phrase in the Search field. A list of topics containing the value entered appears.
