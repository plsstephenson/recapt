Validate anonymous declaration for identified items

Use this function to quickly validate a mail item's S10 identifier against the declaration ID from Kiosk. Once validated, the system generates the ITMATT message, resulting in faster processing time.

You cannot review the declaration data or print the CN23 form using this function. Use the Create declaration for outbound item at post function to perform those actions.

To validate an anonymous declaration for identified items

  1. Enter the item ID or scan the S10 label attached to the mail item, if you have a barcode scanner attached to your workstation.
  2. Enter the declaration ID.
  3. Click Store. CDS performs the validation and generates the ITMATT message.
