Manage users

You use the Manager users function to:

  • add, edit and delete Operator and Administrator users within your organization; there is no limit to the number of users you can add to either of the user groups
  • add, edit and delete user groups

To add a new user

  1. Click Add user.
  2. Enter the Username, Password and full Name of the user. If you check Active Directory user, the password you enter here will be overridden by the Active Directory password for the user.
  3. Choose the user groups to which to add the user by clicking one or more of the corresponding check boxes. There are two default groups for operations and administrator users; you can define others by adding a new user group as described in the procedure below.
  4. Click Store.

To edit or delete an existing user

  1. Click Edit next to the user and follow the steps in the previous section to update the user details, or Delete to delete the user.

To add a new user group

  1. Click Add user group.
  2. Enter the Name of the user group.
  3. Click Store.

After clicking Store, you can review the list of current own user groups defined. You can now add new users to the group by checking it when you add a new user.

To edit or delete an existing user group

  1. Click Edit next to the user group and follow the steps in the previous section to update the group details, or click Delete to delete the user group.
