Manage customs taxes

You use the Manage customs taxes function to configure tax rules that can be applied to inbound or outbound mail item declarations. New tax rules are taken into consideration when calculating the duty and tax assessment of mail items.

If this function does not provide the duties and taxes logic required for your organization, you can capture more complex logic using the scripting functionality available to system administrators. See the CDS Installation and Configuration Guide for more details.

To configure automatic tax calculation rules

  1. Click Add to add a new type of tax to be calculated in the duties and taxes assessment of mail items.

    2. Customs taxes in CDS have three main characteristics, which are displayed as sections in the Add/Edit customs tax screen.

    Section Description
    Type and observations Refer to the Customs type and any relevant observations.
    Formula

    For configuration of the formula that will calculate this type of tax. Currently, this formula is the result of adding:

    • a base value
    • a percentage of the total declaration value
    • a factor of the declaration weight (in grams)
    • a factor of the number of content pieces
    Rules to calculate taxes for inbound/outbound mail For defining a set of rules to determine to which inbound/outbound items the taxes apply (for example, there could be a special tax for alcohol, for items sent to a given province, or for items whose value is above or below a given threshold).
  2. Click Store. Next time you click Manage customs taxes, CDS displays the new tax in the list.

Example

We will now create a special tax for alcohol. It will affect items containing content pieces with the following HS codes: 220421 - wine, 220300 - beer, 220890 - other spirits. The formula for calculating this tax will be: 5 CHF (base tax) + 12.5% of total value + 3 CHF per weight unit (kilo or liter in this case).
